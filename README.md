# Guide to Running a React App TypeScript Project

## Installing Node_Modules

1. **After downloading the project from Github**

Github link:

    https://github.com/ngtutai/DoAn.git

Copy and paste it into your browser to access the Github repository containing this project.

2. **Wait for the download to finish, then reopen the project or use the following command to navigate to the project folder**

```bash
cd NameFolder
```

3. **Install Node_Modules**

Open the **TERMINAL** in Visual Studio Code, run the following command, and wait for the installation to finish:

```bash
npm install bootstrap
```

After the installation is complete, the setup is finished. The only remaining step is to run the command to start the application.

## Command to Run the Application

### Open the TERMINAL in Visual Studio Code

Enter the following command:

```bash
npm start
```

This runs the application in development mode.  
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
